# 🐴 马上发财 - 堆叠消除游戏

一款类似"羊了个羊"的堆叠消除游戏，采用马场主题。

## 🎮 游戏玩法

- 点击卡片进入卡槽，三个相同类型的卡片连成一线即消除
- 卡片分多层堆叠，只有未被上方卡片遮挡的才能点击
- 消除所有卡片即可通关

## ✨ 特色功能

- **3个关卡**：难度递增的挑战
- **4种道具**：撤销、洗牌、移出、提示
- **Web3功能**：
  - MetaMask 钱包连接
  - 代币支付购买道具
  - 通关铸造 NFT 徽章
  - 链上积分排行榜
